# URLs de Swagger UI - Microservicios Fixsy

## Acceso Rápido a Swagger

### Microservicio de Usuarios (Puerto 8081)
**URL Swagger UI:**
```
http://localhost:8081/swagger-ui/index.html
```

**URL API Docs (JSON):**
```
http://localhost:8081/api-docs
```

### Microservicio de Solicitudes (Puerto 8082)
**URL Swagger UI:**
```
http://localhost:8082/swagger-ui/index.html
```

**URL API Docs (JSON):**
```
http://localhost:8082/api-docs
```

### Microservicio de Vehículos (Puerto 8085)
**URL Swagger UI:**
```
http://localhost:8085/swagger-ui/index.html
```

**URL API Docs (JSON):**
```
http://localhost:8085/api-docs
```

## Notas

- Asegúrate de que los microservicios estén corriendo antes de acceder a Swagger
- Si usas un dispositivo físico, reemplaza `localhost` con la IP de tu PC (ej: `http://192.168.1.X:8081/swagger-ui/index.html`)
- Si el error 500 persiste, verifica los logs del microservicio al acceder a `/api-docs`
